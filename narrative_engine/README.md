# 放学之后 · Campus Engine Slice

本仓库仅含 **SCAR / After School** 引擎模块。

```bash
python run_campus.py --gentle --agent-brief --quiet
python run_campus.py --watch-cheng --agent-brief --quiet
python run_campus.py --polite --agent-brief --quiet
```

| 输出 | 说明 |
|------|------|
| `output/campus/events.jsonl` | 真相源 |
| `output/campus/outline.md` | 事件提纲（Agent 自用） |
| `output/campus/agent_brief.json` | 续玩简报 |
