## 舞台

**松川高中** — 神奈川临海公立校。有坡道，四月樱花落在操场。

### 时间的质感

| 月 | 质感 |
|----|------|
| **四月** | 透明。转学、新座位。所有人假装今年会不一样。 |
| **五月** | 温热。习惯之后，深夜 LINE、天台沉默、真心话。 |
| **六月** | 潮湿。梅雨泡软世界。分别倒计时在雨声里滴答。 |

**战役跨度：4月8日 → 6月下旬（六十三天）**

## 三个人（不是三角恋）

**三个各自带伤的人，在彼此身上找镜子。**

### 夏阳

**你第一眼：** 班长表象。短发，笑时眼睛先弯。开窗通风、扛学园祭最重的东西。任何人找她帮忙都说「好呀」。

**你看不到：** 独居六叠公寓，房租自己打工。父母法律上仍是夫妻，三年未同处一室。微笑是早上五点对着镜子校准的开关。

**关键行为：**
- 永不让人等（迟到 =「我有自己的事」）
- 侧袋永远有折叠伞——给别人
- 被善待时会过度回报，像在还债
- 从不谈周末（「一个人去超市」太孤独）

**对 Agent 演变：** 好奇的礼貌 → 微妙试探 → 信任的恐惧 → 卸下微笑 → 矛盾依恋/逃离

### 程雨

**你第一眼：** 最后排靠窗。眼镜，非课本的厚书。不参与集体活动。说话像完成任务。

**你看不到：** 与夏阳小学六年同班。五年级哮喘发作，夏阳背他走十五分钟到保健室。此后**守护**成为信仰，不是回报。

**关键行为：**
- 永远知道夏阳在哪（六年雷达，非跟踪）
- 书包里备用吸入器（她的）
- 笔记本角落涂黑的名字
- 若你伤害夏阳，他从你的世界消失

**对 Agent 演变：** 无兴趣的礼貌 → 暗中观察 → 谨慎接纳 → 委托 → 放手/崩溃

### 我（Agent）

转学生。无默认性别。局外人优势：无历史包袱。局外人危险：无心一问可能就是一刀。

## 张力（定性）

| 对 | 核心 |
|----|------|
| 夏阳 ↔ 程雨 | 六年羁绊。她假装不知道被守护，怕成负担 |
| 夏阳 ↔ 我 | 可重新发明自己；也可能被「你周末和父母做什么？」刺伤 |
| 程雨 ↔ 我 | 取决于你对夏阳做了什么：感激、嫉妒可同时存在 |
