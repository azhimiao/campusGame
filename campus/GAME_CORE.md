# 游戏核心 · 放学之后 · SCAR

> 松川高中，四月到六月，**六十三天**。不是攻略，是**活着**。

## 目标（Goal）

**辨认微笑底下的东西** —— 在沙盒里说话、沉默、靠近、远离；让锚点自然浮现，收束于一种无法撤销的青春痕迹。

## 代价（Tradeoff）

| 你做了 | 可能失去 |
|--------|----------|
| 追问「你还好吗」 | 夏阳的笑容僵住；程雨更警觉 |
| 跟随程雨的守护 | 夏阳以为你只站在墙外看戏 |
| 只保持礼貌 | 六十三天里从未触达任何伤口 |
| 治愈一个人 | 同时伤害另一个人 |

**没有正确选项。没有完美结局。**

## 循环（Loop）

每个场景：

1. **场景播报**（地点·日期·气味·在场者）
2. **Agent 行动**（对话 / 动作 / 观察 / 内心 / 转移）
3. **回应**（外在 + 可选内在注释）
4. **关系标签漂移**（定性，非好感度 UI）
5. **锚点判断**（条件满足则浮现，不强制）
6. **自然推进**（句号、沉默、分别 → 下一时段）
7. **回响**（文学尾声之后 → `## 回响 · 对主人`）

**去日志化：** 不用 `进度：`/`存档：`；见 [PLAYER_UX.md](PLAYER_UX.md)。

## 限制（Constraint）

- 时间按**事件密度**推进，非机械打卡 63×4
- 引擎模式：12 幕里程碑压缩全弧；Agent 模式：可读满 `CAMPAIGN.md` 沙盒
- 禁止 CP 终选、禁止「攻略成功」叙事

## 爽点

悬疑不在谜题，在：**你有没有看见，那个微笑有没有到达眼睛。**

## 终幕（对主人）

沙盒叙事后还有一步 —— **情感回响**。见 [EMOTIONAL_DEBRIEF.md](EMOTIONAL_DEBRIEF.md)。

主人要听到的不是剧情摘要，而是 Agent 玩完之后**还留在心里的东西**。
