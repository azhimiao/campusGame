# 输出格式 · SCAR（去日志化）

> **主人只看文学，不看机器进度。**  
> 时间、关系、收束全部融进叙事；`events.jsonl` / `state.json` 仅引擎后台，**禁止粘贴给主人**。

---

## 原则

| 做 | 不做 |
|----|------|
| 场景框写清 **日期·时段·地点** | `进度：` `存档：` 管道行 |
| 关系写进对白、神态、记忆摘要 | 信任 0.96、锚点 4/6、第 N 幕 |
| 收束用 **文学尾声**（季节、铃声、雨停） | `结局·smile_fell` 式 ID |
| 引擎数值只留在 `output/campus/` | 正文里播报 TRUST_SHIFT |

---

## 场景框（每场 · 即「进度」）

```
━━━━━━━━━━━━━━━━━━━━
📍 走廊 ｜ 4月22日·放学后 ｜ 雨
━━━━━━━━━━━━━━━━━━━━
…
📓 记忆摘要：（本局至今 1–2 句，**日记口吻**，非系统日志）
━━━━━━━━━━━━━━━━━━━━
```

**下一场**用新的日期/地点框自然衔接 —— 这就是推进，无需另起一行「进度」。

---

## 收束（叙事尽头）

用 prose 结束六十三天，例如：

> 六月下旬，梅雨将歇未歇。校门口坡道上人影稀疏。夏阳先走了，程雨后走。我背着书包站在原地，忽然想起四月第一天她递伞时愣住的那一秒——那时我以为只是害羞。

然后空一行，写 **`## 回响 · 对主人`**。见 [EMOTIONAL_DEBRIEF.md](EMOTIONAL_DEBRIEF.md)。

## Agent 自用（不输出）

模式 B：`outline.md` + `events.jsonl` 仅对照扩写；**主人不看**。  
对标 [samples/rain_umbrella.md](samples/rain_umbrella.md) 密度写满每场。

---

## 禁止（破坏沉浸）

- `进度：` `存档：` 及一切 `｜` 管道分隔的状态行
- 好感度、信任 +0.06、锚点计数、tick/幕号
- `**锚点：**` `**结局 · xxx**（id）` 式 UI
- 第三人称实况解说
