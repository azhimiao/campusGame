# 标杆 · 五月深夜 LINE（勿照抄）

> 锚点 γ · 深夜的夏阳和白天的不是同一个人。

---

夜里十一点四十七。宿舍灯早关了，室友的呼吸很匀。我盯着天花板，手机扣在胸口，像压着一颗多余的心跳。

屏幕亮了一下。

夏阳。

不是群组。不是通知。是她。

```
夏阳：睡了吗？
```

我盯着那四个字很久。像怕回复太快会显得我一直在等，又怕太慢了她会当没看见。

```
我：还没。你怎么也没睡？
```

对面显示「正在输入…」，停住，又输入。反复两次。

```
夏阳：……
```

一个省略号。什么都没有。

我打字：「今天谢谢你」——又删掉。我们白天没发生什么值得谢的事。谢什么？谢她在走廊对我笑？谢她假装自己有第二把伞？

最后只发：

```
我：怎么了？
```

很久。久到我以为信号死了。

```
夏阳：[图片]
```

一张天花板。日光灯关着，但角落有充电器的红点。构图歪的，像随手举起来就拍。

没有字。

我：

```
我：你在家？
```

```
夏阳：嗯。
```

```
夏阳：一个人。
```

三个字分开两条发。像每一个字都要单独勇气。

我想问「父母呢」，想起转学填表那天她只写了紧急联系人一栏——空白。把话咽回去。

```
我：要不要……打个电话？我听得见。
```

输入框又显示她在打。然后消失。

```
夏阳：不用啦。
```

```
夏阳：你明天还要早起。睡吧。
```

最后一条之后，头像灰了。

我把手机放回枕头边。窗外有虫鸣。四月说「睡了吗」的人，五月说「睡吧」——是同一张脸，不同的重量。

📓 记忆摘要：
  白天她是班长，笑到眼角。夜里她发一张天花板，说一个人。
  我没问出口的那句，比问了更沉。
