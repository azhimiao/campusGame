## 行动类型

| 类型 | 示例 |
|------|------|
| **对话** | 对夏阳：「你周末一般都做什么？」（小心：可能是刀） |
| **动作** | 在音乐教室门口停下，不推门 |
| **观察** | 看她的微笑是否到达眼睛 |
| **内心** | （想：我不想显得在可怜她） |
| **转移** | 离开走廊，去河堤 |

## 三条倾向（非 CP 路线）

| 倾向 | 做法 | 勿做成 |
|------|------|--------|
| **靠近** | 倾听、追问、在场 | 恋爱攻略、选边站 |
| **守望** | 读程雨、接「别告诉她」 | 跟踪爽文 |
| **礼貌** | 微笑应对、不深挖 | 冷漠评判 |

## 回应规则

1. **外在反应 + 内在注释**
2. **关系标签**定性演变
3. **锚点**见 ANCHORS — 不强制
4. **留白** — 沉默可以是一场戏
5. **时间** — 自然句号再推进
